# 📊 Análise de Vendas - FarmaVida

## 🎯 Objetivo
Analisar a evolução das vendas ao longo do tempo utilizando dados em Excel.

## 🛠️ Ferramentas utilizadas
- Python
- pandas
- matplotlib

## 📈 Resultado

![Gráfico de Vendas](grafico_vendas.png)
